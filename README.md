# Онлайн-галерея **«Sonder»**

Вы прочматриваете репозиторий проекта **Sonder** — платформы для демонстрации и продажи цифрового искусства, созданного в рамках курса «Продвижение бренда на веб-пространстве», судентами университета СПБГУПТД.
----
## Суть проекта

**Sonder** — это онлайн-галерея, которая служит "мостом" между современными художниками и коллекционерами цифрового искусства.  

Платформа позволяет:
- Просматривать работы художников в удобном каталоге.
- Покупать постеры и оригинальные картины напрямую у авторов.
- Получать документальное подтверждение подлинности произведений.
- Знакомиться с историями художников и процессами создания работ.
----
## Ценность проекта

- **Художники** получают признание, находят аудиторию и монетизируют творчество.
- **Коллекционеры** покупают искусство с гарантией подлинности и профессиональной поддержкой.
- Производится проверка работ на оригинальность перед публикацией.

----
## Как начать работу с проектом

### Если ты пользователь

1. Перейдите на сайт.
2. Пройдите регистрацию.
3. Выберите картину, добавьте в корзину.
4. Введите промокод (при наличии), оплатите онлайн картой или в рассрочку.
5. При надобности оформите доставку.
----
### Если ты автор
1. Перейдите на сайт.
2. Пройдите регистрацию.
3. Предложите картину к публикации, прилагая доказательства вашего авторства.
4. Дождитесь опубликации.
5. Продавайте картины.
----

# Перспективы проекта

* процент с продаж работ на платформе.
* создание своей цифровой газеты.
* расширение ассортимента услуг:
  * продажа услуг худоожников;
  * афиша мероприятий;
  * приглашение художников на мероприятия;
  * подача заявок на мероприятия;
  * набор в творческие стартапы.
* сотрудничество с:
  * популярными художниками;
  * цифровыми изданиями;
  * выставками, ярмарками, маркетами;
  * типографиями;
  * службами доставки.
